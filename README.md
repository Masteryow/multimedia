# Multimedia Studio

A static multi-page site for 3D rendering and rotoscope work with built-in video playback.

## Pages

- `index.html` for the home page and featured videos
- `3d-rendering.html` for rendering tutorials
- `rotoscope.html` for rotoscope workflow notes
- `about.html` for publishing notes and site info

## Features

- Navigation menu shared across all pages
- Inline video player on the home page that accepts a direct video URL or a local file
- Responsive layout designed for GitHub Pages

## Publish on GitHub Pages

1. Push these files to a GitHub repository.
2. In GitHub, open the repository settings.
3. Enable GitHub Pages from the main branch.
4. Make sure `index.html` is in the repository root.

## Notes

If you want to host your reel directly inside the page, use an MP4 or WebM file and either:

- place it in the repository and link to the file path, or
- paste a direct file URL into the player.
