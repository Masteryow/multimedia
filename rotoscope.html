1

Import the Reference Footage

Open After Effects and import the reference video clip into your project. Drag it onto the timeline to create a new composition based on the footage settings.

2

Plan Your Body Part Layers

Decide which body parts to isolate — head, torso, arms, legs, etc. Each part gets its own layer so they can be colored and animated independently.

3

Duplicate the Layer & Add a Green Screen Solid

Select the footage layer in the timeline, then press Ctrl + D to duplicate it for each body part you plan to rotoscope. Additionally, create a new Solid layer (Layer > New > Solid…) and set its color to pure green — this acts as your green screen background to make masking and compositing easier.

💡 Rename each duplicated layer clearly (e.g. "Left Arm", "Head", "Torso") and place the green solid at the very bottom of the stack so it shows through wherever the subject hasn't been masked yet.
4

Select the Specific Layer (Body Part) Then Pre-compose

Before pre-composing, make sure you have selected the exact body part layer you want to isolate in the timeline — only that layer should be highlighted. Then pre-compose it to keep the roto work isolated. Use the shortcut:

Ctrl + Shift + C
⌨️ In the Pre-compose dialog, choose "Move all attributes into the new composition" so your masks and effects stay inside the pre-comp. If you don't select the correct layer first, you'll pre-compose the wrong content!
5

Rotoscope Frame by Frame

Double-click into each pre-comp and use the Pen tool or Roto Brush to trace the body part outline on every frame. Adjust the mask path carefully to follow the movement.

6

Refine Matte Edges (Apply Effects to the Pre-comp Layer)

In the main composition, select the pre-comp video layer you just made, then right-click → Effect to add the following effects directly onto that pre-comp layer to clean up the mask edges:

🔲 Fast Box Blur — right-click the pre-comp layer → Effect > Blur & Sharpen > Fast Box Blur to soften hard, jagged edges. Keep the blur radius low (2–5 px) to avoid losing detail.
✂️ Simple Choker — right-click the pre-comp layer → Effect > Matte > Simple Choker to contract or expand the matte boundary. A small positive choke trims stray edge pixels; a negative value expands the matte if it's too tight.
7

Apply Flat Color to Each Body Part

Once a body part is masked, create a new Solid layer with your desired flat color and place it above the pre-comp in the timeline. This creates the signature stylized rotoscope look.

8

Set the Track Matte on the Solid Color Layer

Select the solid color layer you just added. In the timeline, find the Track Matte dropdown (TrkMat column — enable it via the Switches/Modes panel if it's hidden). From that dropdown, select the pre-comp video layer with the rotoscope you want to apply the color to. This clips the solid color precisely to the rotoscoped silhouette.

🎭 Choose "Alpha Matte" from the Track Matte dropdown so the solid color is only visible where the pre-comp's alpha channel is opaque — i.e., exactly the rotoscoped area.
9

Repeat for All Body Parts

Go back to the main composition and repeat Steps 3–8 for every body part layer. Stack the pre-comps and their corresponding color solids in the correct order (arms behind torso, etc.) to build the complete figure.

10

Review, Refine, and Export

Play through the full animation in the main composition. Adjust timing, layer order, and colors as needed, then export via Composition → Add to Render Queue and set your output format and destination before hitting Render.
