Tools & Software

What was used to create these projects

🟠

Blender

Open-source 3D creation suite used for modelling, shading, animation, and rendering the Solar System project.

💡

Cycles Render Engine

Physically-based path tracer in Blender, used with GPU acceleration for realistic lighting and shadows.

🎮

NVIDIA RTX 3050 Ti

GPU used to accelerate Cycles rendering via CUDA compute. OptiX denoising reduced noise without raising sample counts.

🎬

Rotoscoping Software

Frame-by-frame tracing over reference video footage to create the 12-week animation series.

🎞️

FFmpeg / VSE

Used to compile rendered image sequences into MP4 video via Blender's Video Sequence Editor and FFmpeg.

🌐

HTML / CSS / JS

This portfolio site is built with vanilla HTML, CSS, and JavaScript — hosted on GitHub Pages with no build tools.

Portfolio Projects

What's in this site

🪐 3D Rendering

Solar System in Blender

Full modelling-to-render pipeline: UV unwrapping, Principled BSDF shaders, Sun emission lighting, Saturn ring transparency, OptiX denoising, and a Blender Compositor post-process. Rendered at 1920×1080.

 🎬 Rotoscope

12-Week Rotoscope Series

A semester-long rotoscope animation project. Each week produced a new animation clip tracing real footage. All 12 weeks are embedded as YouTube videos, with a reference video and production credits.
